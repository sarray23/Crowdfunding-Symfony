I) HTML Structure - top

Top slider

This template uses Mega Slider for homepage slider. Here is the general structure

<div class="md-slide-items md-slider" id="md-slider-1" data-thumb-width="105" data-thumb-height="70">
	<div class="md-slide-item slide-0" data-timeout="6000">
		<div class="md-mainimg">{SLIDE BACKGROUND IMAGE}</div>
		<div class="md-objects">
			{SLIDE OBJECTS}
		</div>
	</div>
	<div class="md-slide-item slide-1" data-timeout="6000">
		<div class="md-mainimg">{SLIDE BACKGROUND IMAGE}</div>
		<div class="md-objects">
			{SLIDE OBJECTS}
		</div>
	</div>
</div>

We need to include animate.min.css, md_slider.css, md_slider.min.js, jquery.touchwipe.min.js plugin inside the page.

<link rel="stylesheet" href="css/animate.min.css"/>
<link rel="stylesheet" href="css/md-slider.css"/>
<script type="text/javascript" src="js/jquery.touchwipe.min.js"></script>
<script type="text/javascript" src="js/md_slider.min.js"></script>

Use the following script in script.js to fire the plugin

if ($("#md-slider-1").length) {
		$("#md-slider-1").mdSlider({
			fullwidth: true,
			transitions: "fade",
			width: 980,
			height: 365,
			responsive: true,
			slideShowDelay: 6000,
			slideShow: true,
			loop: true,
			showLoading: false,
			showArrow: 1,
			showBullet: 1,
			posBullet: 2,
			showThumb: false,
			enableDrag: true
		});
	}

For more information about this plugin, you can views plugin source code (in js folder). This plugin is available as WordPress plugin and Drupal module too.

II) CSS Files and Structure - top

This theme uses style.css file as main CSS file.

style.css

Main CSS files in this theme is style.css that contains all of specific stylings for the theme.The file is separated into 13 sections using labels as follows:

		
1.	Reset
2.	960 Grid System
3.	Common css (Link, button, margin class, text align class...)
4.	Layout extend
5.	Navigation
6.	Icons
7.	Form
8.	Project teaser
9.	Header
10.	Footer
11.	Home page
12.	Other pages

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

responsive.css

responsive.css contains styling for media queries that allows us to tailor to different resolutions and devices without changing the content. The file is separated into sections as follows:

	
@media screen and (max-device-width:  979px), screen and (max-width:  979px) {
	
{SOME CODES}

}

@media screen and (max-device-width:  767px), screen and (max-width:  767px) {
	
{SOME CODES}

}

@media screen and (max-device-width:  480px), screen and (max-width:  480px) {

{SOME CODES}

}

@media screen and (max-device-width:  320px), screen and (max-width:  320px) {

{SOME CODES}

}

If you would like to edit styling of the specific device width, simply find the appropriate label in the responsive.css file, and then scroll down until you find the appropriate style that needs to be edited.

Additional css files

  1. animate.min - Animation library, from http://daneden.me/animate

III) JavaScript - top

This theme imports following Javascript files.

  1. html5.js - Javascript for IE 9
  2. jquery-1.9.1.min.js - Javascript library
  3. jquery-migrate-1.2.1.min.js - jQuery migrate (require by md_slider.js)
  4. md_slider.min.js - Homepage slider
  5. jquery.touchwipe.min.js - Support touchwipe on homepage slider
  6. jquery.form.js - allows you to upgrade HTML forms to use AJAX
  7. jquery.validate.min.js - validate contact form and comment form
  8. jquery.sidr.min.js - Responsive navigation jQuery plugin
  9. selectnav.min.js - Responsive navigation jQuery plugin
  10. raphael-min.js - A small JavaScript library that should simplify your work with vector graphics on the web
  11. jquery.tweet.min.js - Tweet.js-Mod
  12. script.js - contains all custom script
  13. pie.js - custom script to draw circle percentage

Except script.js and pie.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.

script.js contains custom scripts and scripts to call method for above plugin. The file is separated in to sections with appropriate label.
